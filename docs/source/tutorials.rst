Tutorials
=========

Interactive Examples
--------------------

Explore these Jupyter notebooks to learn how to use the ``retire`` package:

**THEMA Walkthrough**
   A comprehensive guide to using THEMA (topological data analysis) to categorize U.S. coal plants into distinct retirement risk groups.
   
   `View Notebook on GitHub <https://github.com/Krv-Analytics/retire/tree/main/tutorials/1-using_thema.ipynb>`_

**Visualizing Graph Objects**
   Learn how to use the ``Explore`` class to create visualizations of coal plant network data.
   
   `View Notebook on GitHub <https://github.com/Krv-Analytics/retire/tree/main/tutorials/2-using_explore.ipynb>`_

**Target Matching**
   Understand how to analyze and match target retirement strategies.
   
   `View Notebook on GitHub <https://github.com/Krv-Analytics/retire/tree/main/tutorials/3-target_matching.ipynb>`_

.. note::
   These notebooks are available in the ``tutorials/`` directory of the repository.
